vec3.crossProduct (v1, v2)
Computes the cross product vector of two vectors. The parameters are not commutative.

Parameters

  • v1: The first vector
  • v2: The second vector

Usage

  • local normal = vec3.crossProduct(v1, v2)
  • local normal = v1:crossProduct(v2)
vec3.dotProduct (v1, v2)
